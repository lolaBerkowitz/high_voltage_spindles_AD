# high_voltage_spindles_AD
Repo is intended to identify and examine features of high voltage spindles observed in rat lfp data. Code is in development.

### Dependencies (Python >= 3.5)
* bycycle  = 1.0.0
* neurodsp >= 2.1.0
* numpy >= 1.18.5
* scipy >= 1.4.1
* pandas >= 0.25.3
* matplotlib >= 3.0.3
* pytest (optional)
